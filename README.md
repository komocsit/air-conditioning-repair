# CoolBreeze AC Repair — Next.js

A Next.js static site for CoolBreeze AC Repair, offering professional air conditioning services.

## Pages

| Route | Description |
|---|---|
| `/` | Homepage |
| `/air-conditioning-repair` | AC Repair & Diagnostics |
| `/ac-installation` | AC Installation |
| `/ac-maintenance` | AC Maintenance |
| `/duct-cleaning` | Duct Cleaning |
| `/emergency-ac-service` | Emergency AC Service |
| `/about-us` | About Us |
| `/contact` | Contact |
| `/all-services` | All Services |
| `/blog` | Blog |

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
